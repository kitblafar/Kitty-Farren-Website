Project: Crochet Stegosaurus

I had previously made an African Flower Triceratops and Giant Tarantula so I descided to make one final dino as a gift for a friend. My friend is a dinosaur lover so when I saw this stegosaurus on pintrest I thought Id do one last one. The dino needs to be red-green colour-blind friendly so I chose yellow, blue and purple as the base colours.

Yarns

This project is made from Paintbox Aran Cotton Yarn so its nice and soft. The number of balls is not yet finalised.

  • Dark Aubergine (649) x2
  • Pale Lilac (646) x1
  • Pansy Purple (648) x1
  • Sailor Blue (640) x3
  • Summer Sky Blue (663) x1
  • Daffodil Yellow (622) x1
  • Glorious Gold (666) x2

Motifs

The pattern for a basic 6 sided african flower motif from Janean Trent at Crochet Blog is shown here.

There are various motifs to this design. In order they are 8,7,6,5 and 4 petal african flow motifs and follow the same pattern as the above design.

The following is a map of where each motif should go in the dino and the total number of each design is listed below. The next image is the basic dino outline- he's massive!!! Next to be added are some eyes and the back fins.
